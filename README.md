# TerraGuard

TerraGuard is a modern, real-time mobile application designed to solve a critical world problem: **Disaster Awareness and Response**. By tracking real-time seismic activity globally, it helps users stay informed about natural disasters as they happen, ensuring timely awareness without relying on third-party aggregators or delayed news cycles.

## Features
- **Real-Time Global Feed:** Live earthquake data fetched directly from the US Geological Survey (USGS) public feeds.
- **No Mock Data:** 100% real, live production data.
- **Modern UI/UX:** Clean typography, status-colored intensity indicators, and smooth animations.
- **Interactive Details:** View exact coordinates, times, magnitude significance, and direct links to official USGS science reports.
- **Cross-Platform:** Beautifully optimized for mobile devices, tablets, and the web.

## Tech Stack
- **Framework:** Flutter (Dart)
- **Data Source:** USGS Earthquakes Live GeoJSON API
- **Design:** Material 3, Google Fonts, Flutter Animate
- **Backend:** Completely Serverless (Direct client-to-API communication, no Node.js intermediary required)

## Getting Started

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the application.
